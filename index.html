2D game collection, 2013

The project itself served as a context for games. Games would be compiled as .libs and then included in the application project, which linked the engine and the game together. Games ran on a separate thread from the engine to prevent bad game code from freezing the application. The project contained two games: a space shooter and a minesweeper clone.

Language: C#. Engine: Custom XNA based. Platform: Windows PC.

Shaders

Voxel experiments, 2014-2015

Here are a couple of different demos I made while learning about procedural terrain. The first one here is not very fast, it was written entirely on the cpu. Here I wrote a hardware accellerated voxel renderer which was much, much better. By encoding voxel data into textures, I could build voxel shapes in a geometry shader at extremely high speeds.



Language: C# and Cg. Platform: Unity.

terrain maps on the GPU, 2015

Demonstrates the power of shader programs to create realtime interactive terrain environments. Takes advantage of Shader model 5.0, using a geometry shader to add foliage to the scene. Height displacement, texturing, additional geometry are all done according to a single texture map input. Click for more information & interactive demo.



Language: C# and Cg. Platform: Unity.
